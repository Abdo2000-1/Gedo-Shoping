
# 🛒 Gedo Shopping Project

![Project Status](https://img.shields.io/badge/Status-Completed-success)
![Platform](https://img.shields.io/badge/Platform-Web-blue)
![Language](https://img.shields.io/badge/Main_Language-JavaScript-yellow)

مشروع **Gedo Shopping** هو تطبيق ويب للتجارة الإلكترونية يهدف إلى تقديم تجربة تسوق سلسة وبسيطة، مع واجهة مستخدم متجاوبة ولوحة تحكم متكاملة.

---

## 🚀 التقنيات المستخدمة (Tech Stack)

| المجال | الأدوات والتقنيات |
| :--- | :--- |
| **Front-End** | HTML5, CSS3, JavaScript (ES6+) |
| **Styling** | Bootstrap / CSS Custom Properties |
| **Back-End Services** | Firebase (Authentication & Firestore) |
| **Version Control** | Git & GitHub |

---

## ✨ المميزات الرئيسية (Features)

* **نظام تسوق متكامل:** استعراض المنتجات وإضافتها إلى عربة التسوق.
* **لوحة تحكم (Admin Dashboard):** لإدارة المنتجات، الطلبات، والمستخدمين.
* **التجاوب (Responsive Design):** تصميم متوافق مع جميع أحجام الشاشات (موبايل، تابلت، ديسكتوب).
* **Firebase Integration:** استخدام قاعدة بيانات حقيقية لتخزين البيانات وضمان سرعتها.
* **واجهة مستخدم عصرية:** الاعتماد على تجربة مستخدم (UX) مريحة وبسيطة.
